
A simple, modular command-line Library Management System built using Python and Object-Oriented Programming.
It allows users to add, issue, return, view, and search books, with persistent storage using a text file.

This project demonstrates:
Object-Oriented Programming (classes, methods, encapsulation)
Modular coding with packages
File handling for persistent book storage (books.txt)
Search operations (by title, ISBN)
CLI menu-based application design

Features:
Add new books
Issue books
Return book
Search books by titl
Search books by ISBN
View all book records
Data stored persistently in a text file
Clean, modular codebase
